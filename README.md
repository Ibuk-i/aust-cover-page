# AUST Lab Report Cover Page Generator

Open `/tmp/workspace/Ibuk-i/aust-cover-page/index.html` in a browser to use the generator.

It runs fully offline and provides:
- Department toggle (EEE automated mode / manual department)
- Course + experiment presets with title override
- Editable final print values for course/experiment info
- Student identity, term, and date fields with live A4 preview
- Print/PDF button with auto file name base: `[Course Code]- Exp [Exp No]- Cover Page [Auto Generated]`
